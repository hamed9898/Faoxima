# 📚 Faoxima Wiki

> **🇮🇷 فارسی** | [🇬🇧 English](#-english)

ربات و پنل فروش/مدیریت VPN مبتنی بر **PHP 8.2 + MySQL** با پشتیبانی از ده‌ها پنل، درگاه‌های پرداخت ایرانی و کریپتو، **سیستم نمایندگان چند‌مستأجری**، **ربات اختصاصی هر نماینده**، **پشتیبانی پراکسی برای هاست ایران** و **مرکز کنترل فروش مدیر**.

این ویکی تمام آموزش‌های لازم برای نصب، تنظیم و استفاده از سورس را به‌صورت گام‌به‌گام پوشش می‌دهد.
